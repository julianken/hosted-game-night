# Ball Rolling Sound Effects

This directory contains sound effects for the bingo ball rolling animation.

## Required Files

| File | Description |
|------|-------------|
| `roll.mp3` | Clean ball rolling sound (1-3 seconds, blower-style preferred) |
| `roll-hall.mp3` | Same sound with hall reverb effect |

## Audio Specifications

- **Duration:** 1-3 seconds (ideal: 2 seconds)
- **Format:** MP3, 128-192 kbps, stereo
- **Volume:** Normalized to ~-14 LUFS
- **Style:** Blower-style (air-powered) for authentic bingo hall feel

## Recommended Sources

- **Epidemic Sound:** "Bingo, Miniature, Balls, Roll In Plastic Cage" (trim to 2 sec)
- **Pond5:** 194+ bingo ball sounds (starting at $2)
- **AudioJungle:** Bingo sound packs (320kbps, 16-bit stereo)

## Creating Hall Variant

1. Start with `roll.mp3`
2. Apply hall reverb effect (large room, ~1.5s decay)
3. Export as `roll-hall.mp3`

Tools: Audacity (free), Adobe Audition, Logic Pro
